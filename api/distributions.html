class maoud.distributions.AlphaMu(alpha, mu)[source]¶

Defines the α — μ probability distribution. For the theorectical aspects of the α — μ distribution, see M. D. Yacoub, The α — μ distribution: A physical fading model for the Stacy distribution, IEEE Trans. Veh. Technol., vol. 56, no. 1, pp. 27–34, 2007.

Attributes

alpha, mu (float, float) Parameters that define the α — μ distribution.
